# Java HandsOn Project

This repository contains hands-on Java programming exercises covering:

- Basic arithmetic, patterns, loops
- Factorial, Fibonacci (iterative/recursive)
- Palindrome check, leap year, vowel count
- Coin change DP, power, Armstrong numbers
- Patterns (square, rombus)
- And more (see all .java files)

## How to compile/run

```bash
javac *.java
java ClassName
```

Ignore .class files (gitignored).
